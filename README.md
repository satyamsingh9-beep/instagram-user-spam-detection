# 🛡️ Instagram Spam Detector

An AI-powered web application that detects spam comments and messages using Machine Learning and Natural Language Processing (NLP). This project helps identify unwanted promotional, fraudulent, and suspicious content in real time, improving the quality of online interactions.

## 📌 Project Overview

Social media platforms receive millions of comments and messages every day. A significant portion of this content consists of spam, scams, phishing attempts, and promotional messages.

The Instagram Spam Detector leverages Machine Learning and NLP techniques to automatically classify text as either:

✅ Not Spam

❌ Spam

This application allows users to enter comments or messages and instantly receive a prediction.

## 🧠 How It Works

### Step 1: Input

The user enters an Instagram comment or message.

### Step 2: Text Preprocessing

The application processes the text by:

* Converting text to lowercase
* Removing punctuation
* Removing special characters
* Cleaning unwanted text patterns

### Step 3: Feature Extraction

The cleaned text is transformed into numerical features using NLP techniques.

### Step 4: Prediction

The Machine Learning model analyzes the features and predicts whether the content is:

* Spam
* Not Spam

### Step 5: Result Display

The prediction result is displayed instantly to the user.
