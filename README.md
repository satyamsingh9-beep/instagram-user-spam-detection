# Instagram Spam Detector

An AI-powered web application that detects spam comments and messages using Machine Learning and Natural Language Processing (NLP).

## Features
- Real-time spam detection
- User-friendly interface
- Machine Learning based classification
- Deployed on Replit

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NLTK
- Replit
